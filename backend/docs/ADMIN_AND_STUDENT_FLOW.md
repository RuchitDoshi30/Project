# 🔄 Admin & Student Working Flow  
## Placement Preparation Platform for Engineering Students

This document summarizes the operational flow of both user roles.

---

## Student Flow
1. Login using college credentials
2. Access dashboard
3. Practice coding problems
4. Attempt aptitude tests
5. View progress and results

---

## Admin Flow
1. Login as admin
2. Create/manage student accounts
3. Manage coding problems
4. Manage aptitude questions
5. Review coding submissions

---

## Conclusion

Clear separation of responsibilities ensures smooth operation and easy system management.
